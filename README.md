# TCP-UP - Private Assets Repository

Ce dépôt est une extension de stockage pour le projet **tcp-up.org**. Il fait office de serveur de ressources externe (CDN) afin de ne pas encombrer l'hébergement mutualisé OVHcloud.

## ⚙️ Fonctionnement interne

Les fichiers stockés ici sont appelés de manière distante. Cela permet de conserver un site léger et rapide à déployer sur le serveur principal.

## 📁 Organisation des dossiers

* **`/blog`** : images d'illustration pour les articles de blog.
* **`/img`** : ressources de l'interface (iconographie, logos).
* **`/docs`** : documents PDF et ressources à télécharger.
* **`/kits`** : kits complet à télécharger.
